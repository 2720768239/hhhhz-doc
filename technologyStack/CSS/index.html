1. 属性选择器：[] ​

css
/* 给所有外链添加图标 */
[href^='https'] {
  padding-right: 20px;
  background: url(external-link.svg) no-repeat right center/16px 16px;
}

/* 给所有pdf文件统一颜色和加粗 */
[href$='.pdf'] {
  color: #f00;
  font-weight: bold;
}

/* 给素有搜索框统一样式 */
[type='search'] {
  padding-left: 24px;
  background: url(search.svg) no-repeat left center/16px 16px;
}

2. 结构伪类选择器：:first-child、:last-child、:nth-child(n) ​

css
/* 第一个：first-child */
/* 列表第一项去掉上边距 */
.list-item:first-child {
  margin-top: 0;
}

/* 最后一个：last-child */
/* 列表最后一项去掉下边距 */
.list-item:last-child {
  margin-bottom: 0;
}

/* 指定第几个元素：nth-child(n) */
/* 列表第2项去掉左边距 */
.list-item:nth-child(2) {
  margin-left: 0;
}
/* 奇数：2n+1，偶数：2n */
tr:nth-child(2n) {
  background: #f5f5f5;
}

3. 组合选择器：+、~、> ​

css
/* 紧邻兄弟选择器：选择label后面紧跟的input */
label + input {
  margin-left: 8px;
}

/* 普通兄弟选择器：选中所有的兄弟元素 */
h2 ~ p {
  color: #f00;
}

/* 子元素选择器：只选择直接子元素 */
nav > a {
  display: inline-block;
  padding: 8px 16px;
}

4. 伪类选择器：:hover、:focus、:checked ​

css
/* 按钮悬浮 */
.button:hover {
  transform: translateY(-2px);
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}
/* 输入框聚焦状态 */
.input:focus {
  border-color: #f00;
  box-shadow: 0 0 0 2px rgba(255, 0, 0, 0.2);
}
/* 被选中的单选/复选框文本 */
input[type='radio']:checked + label {
  color: #f00;
  font-weight: bold;

5. 否定伪类选择器：:not() ​

排除特例，不希望被选中的元素

css
/* 除了最后一个元素外，都加分割线 */
.list-item:not(:last-child) {
  border-bottom: 1px solid #ccc;
}
/* 除了禁用的按钮外，都加悬浮效果 */
.button:not(:disabled):hover {
  background: #0066ff;
}

7. 目标伪类选择器：:target ​

页面内导航利器，当 url 有锚点时，可以利用:target突显

css
/* 被导航到的章节添加高亮 */
.section:target {
  animation: highlight 1s ease;
  background: #fff8dc;
}
/* 目录跳转后的标题特效 */
h2:target::before {
  content: '#';
  margin-right: 8px;
}
